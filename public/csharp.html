C#

C# — це сучасна, об'єктно-орієнтована мова програмування, розроблена корпорацією Microsoft. Вона є частиною платформи .NET і дозволяє створювати різноманітні додатки: від веб-сервісів і настільних програм до мобільних додатків та ігор. C# зручна для початківців завдяки зрозумілому синтаксису, а її потужний функціонал робить її популярною серед досвідчених розробників.

Основна мета C# — бути простою та продуктивною мовою, здатною конкурувати з іншими сучасними мовами програмування, такими як Java і C++. Вона чудово підходить для створення корпоративних рішень, а також використовується для розробки веб-додатків з використанням ASP.NET, хмарних сервісів, ігор на платформі Unity та багатьох інших видів програмного забезпечення.

Вступ до C#

C# (Сі-Шарп) — це об'єктно-орієнтована мова програмування, розроблена корпорацією Microsoft на базі платформи .NET. Вона поєднує в собі простоту та продуктивність, а також пропонує високий рівень безпечності типів. Завдяки цим якостям C# підходить як для початківців, так і для досвідчених розробників.

Мову використовують для створення настільних програм, веб-сервісів, мобільних додатків і навіть ігор (через популярний рушій Unity). Середовище розробки зазвичай базується на Visual Studio або більш легкому Visual Studio Code, але є й інші варіанти. Нижче розглянемо основні можливості C# та типові елементи синтаксису.

Особливості C#

  • Об'єктно-орієнтоване програмування (ООП). C# базується на ООП, що означає, що основою будь-якої програми є об'єкти та класи, які представляють реальні або абстрактні сутності.
  • Безпечність типів (type safety). C# забезпечує високий рівень безпеки типів даних, що допомагає уникати багатьох помилок під час компіляції та виконання програми.
  • Інтеграція з .NET. Оскільки C# є частиною екосистеми .NET, програми на C# можуть використовувати потужний набір бібліотек та інструментів для розробки.
  • Гарна підтримка асинхронного програмування. C# дозволяє легко реалізовувати асинхронні операції, що є важливим для створення високонавантажених додатків.
  • Універсальність. Ви можете використовувати C# для розробки як настільних, так і веб, мобільних додатків, а також ігор (наприклад, у Unity).

Початок роботи з C#

Для того щоб почати програмувати на C#, необхідно мати інтегроване середовище розробки (IDE), таке як Visual Studio або більш легку версію — Visual Studio Code. Обидва інструменти підтримують платформу .NET і надають можливості для компіляції та відладки коду на C#.

Після встановлення середовища ви можете створити новий проєкт (наприклад, «Console App»), де і писатимете свій C#-код. Далі система збирає все у виконуваний файл або бібліотеку, яку можна запустити під .NET-оточенням.

Привіт, світ!

Перший код, який ми напишемо на C#, — це простий приклад, що виводить "Привіт, світ!" на екран. Ось як це виглядає: 


using System;

class Програма
{
    static void Main()
    {
        Console.WriteLine("Привіт, світ!");
    }
}

using System; — це директива, що дозволяє використовувати функціонал бібліотеки System.
class Програма — це оголошення класу. У C# все знаходиться всередині класів.
static void Main() — це точка входу в програму, де починається її виконання.

Основи синтаксису C#

Змінні та типи даних

C# є строго типізованою мовою програмування, що означає, що кожна змінна має чітко визначений тип даних. Основні типи даних включають:

  • Цілі числа: int — 32-бітове ціле число, long — 64-бітове ціле число.
  • Числа з плаваючою комою: float, double.
  • Логічні значення: bool — може мати значення true або false.
  • Рядки: string — для зберігання тексту.
  • Масиви: int[] масив = new int[5]; — масив цілих чисел.

Приклад оголошення змінних:


int вік = 25;
string ім'я = "Олексій";
bool єПовнолітній = true;

Console.WriteLine($"Мене звати {ім'я}, мені {вік} років.");

Змінні в C# можуть бути локальними (оголошені всередині методу) або полями класу (якщо оголошені поза методами, але всередині класу). Також у C# існують var (не плутати з JavaScript), які дозволяють компілятору виводити тип змінної автоматично, але він усе одно залишається статично визначеним під час компіляції.

Умовні оператори

Умовні оператори допомагають виконувати певний код залежно від умов. Основні умовні оператори в C#:

  • if: Перевіряє умову.
  • else: Виконується, якщо умова в if не виконується.
  • else if: Використовується для перевірки додаткових умов.

Приклад використання умовних операторів: 


int вік = 18;

if (вік >= 18)
{
    Console.WriteLine("Ви повнолітній.");
}
else
{
    Console.WriteLine("Ви неповнолітній.");
}

Цикли

Цикли в C# дозволяють виконувати одні й ті ж операції кілька разів. Існує кілька типів циклів:

  • for: Використовується для повторення дій певну кількість разів.
  • while: Виконує код, поки умова залишається істинною.
  • foreach: Призначений для ітерації по колекціях, таких як масиви або списки.

Приклад циклу for:


for (int i = 0; i < 5; i++)
{
    Console.WriteLine($"Номер {i}");
}

Цикл foreach використовується для обходу елементів колекцій (наприклад, List<T>), і має синтаксис: foreach (var елемент in колекція) { ... }. Це полегшує роботу з колекціями, оскільки не потрібно керувати індексами.

Функції (методи)

Функції або методи — це блоки коду, які можна викликати для виконання певних дій. Методи можуть приймати параметри та повертати результат. 


int Сума(int a, int b)
{
    return a + b;
}

int результат = Сума(5, 3);
Console.WriteLine($"Результат: {результат}");

Якщо метод не повертає значення, вживають void. Використання методів допомагає розбити програму на логічні частини і уникнути дублювання коду.

Об'єктно-орієнтоване програмування (ООП) в C#

C# є об'єктно-орієнтованою мовою програмування, що означає, що основна ідея цієї мови — це робота з об'єктами. Клас — це шаблон (креслення) для створення об'єктів. Об'єкти мають властивості (дані) та методи (дії).

Класи та об'єкти

Клас у C# можна порівняти з кресленням автомобіля, а об'єкт — це конкретний автомобіль, створений за цим кресленням. Клас визначає структуру об'єкта, а об'єкт є його екземпляром.

Ось приклад створення класу:


class Автомобіль
{
    public string Марка { get; set; }
    public string Модель { get; set; }

    public void Опис()
    {
        Console.WriteLine($"Це {Марка} {Модель}.");
    }
}

Створення об'єкта на основі класу:


Автомобіль авто = new Автомобіль();
авто.Марка = "Tesla";
авто.Модель = "Model S";
авто.Опис();

Інкапсуляція

Інкапсуляція означає приховування внутрішньої реалізації об'єкта та надання доступу до даних через методи або властивості. Це робить код більш безпечним і гнучким. 


class Книга
{
    private string назва;

    public string Назва
    {
        get { return назва; }
        set { назва = value; }
    }
}

Тут поле назва є закритим (private), а доступ до нього здійснюється через властивість Назва (public).

Спадкування

Спадкування дозволяє створювати нові класи на основі існуючих, що забезпечує повторне використання коду. 


class Тварина
{
    public void Голос()
    {
        Console.WriteLine("Тварина видає звук");
    }
}

class Собака : Тварина
{
    public void Гавкати()
    {
        Console.WriteLine("Собака гавкає");
    }
}

Поліморфізм

Поліморфізм дозволяє використовувати один і той самий метод для різних типів об'єктів. У C# для цього застосовується ключове слово virtual у базовому класі та override у похідному. 


class Тварина
{
    public virtual void Голос()
    {
        Console.WriteLine("Тварина видає звук");
    }
}

class Собака : Тварина
{
    public override void Голос()
    {
        Console.WriteLine("Собака гавкає");
    }
}

Так можна написати метод Голос() в базовому класі Тварина, а потім реалізувати різну поведінку для Собака, Кішка тощо.

Підсумок

C# — це потужна та гнучка мова програмування, яка дозволяє розробляти широкий спектр додатків. Від простих настільних програм до складних корпоративних систем і відеоігор на базі Unity, C# є універсальним інструментом для розробників. Завдяки підтримці об'єктно-орієнтованих принципів, таких як інкапсуляція, спадкування та поліморфізм, C# допомагає створювати зрозумілий, підтримуваний та масштабований код.

Зрозумівши базові концепції C#, ви можете надалі вивчати фреймворки (ASP.NET для вебу, Xamarin для мобільних застосунків), розробляти ігри або працювати з базами даних. Мова та платформа .NET мають величезну екосистему, що відкриває безліч можливостей у розробці.
